# ginit-plus
'git init' on more steriods
## About 

Ginit-Plus Is 'git init' but on steriods,it setups up everything for a git repository


## Requirements

* [Node.js](http://nodejs.org/)
* [Git](https://git-scm.com/)
* [GitHub account](https://github.com/)

## Installation Steps
1.Run `npm i -g ginit-plus`
2.Run `ginit <repo-name> <repo description>`

OR

1. Clone repo
2. Run `npm install`
3. Install the module globally with `npm install -g` from within the project directory
4. Run `ginit <repo-name> <repo description>`
